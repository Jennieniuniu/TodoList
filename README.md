# 功能：

## 1.添加任务模块

1.点击添加新任务，出现任务输入框

2.点击确定或按下回车，添加任务到待办事项

3.输入任务，点击取消，输入框里的文字清空

******************************************

## 2.待办事项模块

1.移入鼠标，出现功能图标

2.完成图标，待办事项移入已完成模块

3.删除图标，删除待办事项

4.水滴图标，选择待办事项的背景颜色

5.星星图标，给待办事项标星

6.双击编辑

## 3.已完成事项模块

1.已完成事项背景颜色变灰

2.移入鼠标，出现功能图标

3.删除图标，删除任务

# 4.撤回图标，已完成事项撤回到待办事项模块
